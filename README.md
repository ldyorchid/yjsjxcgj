# Vue 2 简单项目

这是一个不使用脚手架的 Vue 2 基础项目，只包含了 Vue 2 的核心文件和基本示例。

## 项目结构

```
├── index.html  # 主页面文件
├── js/         # JavaScript 目录
│   └── vue.min.js  # Vue 2 核心文件
└── README.md   # 项目说明文档
```

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件即可查看效果
2. 无需安装任何依赖或运行任何命令

## 功能示例

这个示例包含了 Vue 2 的基本功能演示：
- 数据绑定
- 事件处理
- 列表渲染
- 表单输入绑定

## 注意事项

- 这是一个最简单的 Vue 2 项目，适合快速学习和演示
- 对于实际开发项目，建议使用 Vue CLI 或 Vite 等工具构建完整项目
